# reverse-string

A CLI tool written in C that takes user input and prints the string in reverse.

## Compile

```bash
gcc -o reverse-string reverse-string.c
```

## Usage

```bash
./reverse-string
```

## Example

```
Please enter text: We like to code in C
C ni edoc ot ekil eW
```

## What I learned

- Array indexing and backwards traversal
- String handling with fgets and null terminator management
- strlen for calculating string length
- Buffer sizing (array size vs index position)
